# Navigation and Dropdown Menu Demo

## Overview
This project is an **HTML page** created in the first year of learning web development. It demonstrates **navigation menus and dropdown functionality** using **HTML and CSS**. The page highlights basic concepts of **menu structuring, links, and interactive dropdowns**.

## Features
- Simple **navigation bar** with multiple links.  
- **Dropdown menus** for nested navigation items.  
- Basic **CSS styling** for hover effects and layout.  
- Fully static HTML project – no backend required.  

## Technologies Used
- **HTML** – structure and content of the page.  
- **CSS** – styling and dropdown menu effects.
